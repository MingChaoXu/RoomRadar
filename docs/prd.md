# 酒店竞品价格爬虫工具 PRD

更新时间：2026-03-22

## 1. 产品定义

面向酒店运营团队、收益管理团队的竞品价格监测工具。系统围绕目标酒店自动发现周边竞品酒店，并在携程、美团等渠道持续采集价格，输出竞品价格趋势、最低价和价差情报。

## 2. 产品目标

### 2.1 核心目标

1. 帮助酒店快速识别周边竞品酒店。
2. 帮助收益经理掌握竞品价格变化。
3. 为调价、活动和渠道策略提供数据依据。

### 2.2 非目标

1. 不做在线下单。
2. 不做完整 PMS/CRS 能力。
3. 第一版不做复杂收益管理算法。

## 3. 目标用户

1. 酒店单店运营
2. 酒店集团运营
3. 收益经理
4. 商业分析人员

## 4. 业务流程

1. 用户录入目标酒店。
2. 系统通过高德定位并生成周边竞品候选列表。
3. 用户确认竞品池。
4. 系统定时采集携程、美团价格。
5. 系统做房型映射和价格标准化。
6. 用户查看报表、趋势和异常告警。

## 5. 功能需求

### 5.1 酒店管理

1. 新增目标酒店
2. 编辑酒店基础信息
3. 维护酒店标签

### 5.2 竞品管理

1. 自动推荐竞品
2. 手动增删竞品
3. 设置竞品半径和筛选规则

### 5.3 价格采集

1. 支持携程
2. 支持美团
3. 支持未来 N 天采集
4. 支持每天固定时段采集
5. 支持失败重试和状态查看

### 5.4 数据标准化

1. 酒店标准化匹配
2. 房型标准化映射
3. 价格口径统一
4. 币种统一

### 5.5 报表分析

1. 竞品最低价榜单
2. 平台价差分析
3. 日期趋势图
4. 同圈层均价分析
5. 异常波动告警

### 5.6 导出与通知

1. Excel 导出
2. 邮件日报
3. 企业微信或钉钉告警

## 6. MVP 范围

### 6.1 必做

1. 单酒店录入
2. 高德周边竞品发现
3. 携程价格采集
4. 美团价格采集
5. 日级价格快照
6. Web 看板
7. Excel 导出

### 6.2 延后

1. 多租户
2. 权限系统细粒度控制
3. 调价建议模型
4. 多平台扩展到飞猪、同程、去哪儿

## 7. 关键指标

1. 竞品识别准确率 > 80%
2. 采集成功率 > 90%
3. 价格标准化可比率 > 70%
4. 日报生成时间 < 15 分钟

## 8. 关键页面

1. 酒店列表页
2. 酒店详情页
3. 竞品池管理页
4. 采集任务页
5. 价格对比看板
6. 告警列表页

## 9. 验收标准

1. 输入一家酒店后，系统能自动返回周边候选竞品。
2. 对已确认竞品，可返回携程、美团指定日期的价格快照。
3. 用户可查看至少 7 天维度的趋势数据。
4. 用户可导出结构化 Excel。
